SimpleTXT 隐私政策SimpleTXT Privacy Policy

最后更新日期：2026 年 5 月 17 日 Last Updated: May 17, 2026

SimpleTXT（以下简称"本应用"）是一款本地 TXT 文件阅读工具。我们非常重视您的隐私与数据安全。本隐私政策旨在说明本应用的隐私处理方式。

SimpleTXT (hereinafter referred to as "the App") is a local TXT file reading tool. We value your privacy and data security. This privacy policy explains how the App handles your privacy.

使用本应用即表示您同意本隐私政策的内容。By using this App, you agree to the contents of this Privacy Policy.

1. 我们收集的信息1. Information We Collect

1.1 本应用不收集任何个人信息1.1 The App Does Not Collect Any Personal Information

本应用不会主动收集或上传任何可识别个人身份的信息，包括但不限于：

The App does not actively collect or upload any personally identifiable information, including but not limited to:

  • 姓名
  • Name
  • 电话
  • Phone Number
  • 邮箱
  • Email Address
  • 位置信息
  • Location Information
  • 身份证件信息
  • Identity Document Information
  • 不会主动收集 IMEI、Android ID 等系统级设备标识符
  • System-level device identifiers such as IMEI or Android ID

我们不会收集您的个人身份信息，不进行用户画像或广告投放。为了改进应用体验，我们通过 Firebase Analytics 收集匿名使用统计数据（详见第 5 条）。

We do not collect personally identifiable information, nor do we perform user profiling or ad targeting. To improve the app experience, we collect anonymous usage statistics through Firebase Analytics (see Section 5 for details).

1.2 文件内容1.2 File Contents

无论是：

Whether:

  • 您通过系统文件选择器（SAF）导入的 TXT 文件
  • TXT files imported via the system file picker (SAF)
  • 通过 Wi-Fi 传输导入的 TXT 文件
  • TXT files imported via Wi-Fi transfer

本应用均不会上传、存储或分析这些文件内容。

The App will NOT upload, store, or analyze these file contents.

文件内容仅用于本地阅读，不会离开您的设备。

File contents are only used for local reading and never leave your device.

2. 数据的使用与处理方式2. How Data Is Used and Processed

2.1 本地处理2.1 Local Processing

无论是阅读、解析、换页、还是设置调整，本应用所有核心功能均在设备本地运行。除 Firebase Analytics 发送的匿名统计请求外，本应用不会将您的文件内容或个人信息传输至任何远程服务器。

All core functions of the App—reading, parsing, page turning and settings adjustments—run locally on your device. Except for anonymous analytics requests sent through Firebase Analytics, the App does not transmit your file contents or personal information to any remote servers.

2.2 Wi-Fi 传输模式的数据处理2.2 Data Processing in Wi-Fi Transfer Mode

当使用 Wi-Fi 导入文件时：

When using Wi-Fi to import files:

  • 文件通过您的局域网在电脑与手机之间点对点传输
  • Files are transferred point-to-point between your computer and phone via your local network
  • 数据仅在您的设备与局域网内部流动
  • Data only flows within your devices and local network
  • 除 Firebase Analytics 的匿名统计请求外，本应用不连接外部服务器或上传您的文件内容
  • Except for anonymous analytics requests via Firebase Analytics, the App does not connect to external servers or upload your file contents

我们不记录、不保存、不共享 Wi-Fi 导入过程中的任何信息。

We do not record, save, or share any information during the Wi-Fi import process.

3. 数据共享3. Data Sharing

本应用不会与任何第三方共享您的个人身份信息或文件内容。

The App does not share any personally identifiable information or file contents with any third parties.

我们通过 Firebase Analytics（Google）收集匿名使用统计数据，具体内容详见第 5 条。该数据不包含任何可识别您个人身份的信息。

We collect anonymous usage statistics via Firebase Analytics (Google) as described in Section 5. This data does not include any personally identifiable information.

我们绝不共享以下信息：

We never share:

  • 您的 TXT 文件内容
  • Your TXT file contents
  • 姓名、邮箱、电话等个人身份信息
  • Personal identity information such as name, email, or phone number
  • 精确位置信息
  • Precise location information

本应用遵循最新的 Android 隐私权限规范。

The App complies with the latest Android privacy permission standards.

4.1 文件访问权限（无需传统存储权限）4.1 File Access Permissions (No Traditional Storage Permissions Required)

本应用不会申请 READ_EXTERNAL_STORAGE 或 WRITE_EXTERNAL_STORAGE 权限。

The App does not request READ_EXTERNAL_STORAGE or WRITE_EXTERNAL_STORAGE permissions.

本应用通过 Storage Access Framework（SAF）访问文件：

The App accesses files through the Storage Access Framework (SAF):

  • 文件由您通过系统文件选择器手动选取
  • Files are manually selected by you through the system file picker
  • 应用只能访问您明确选择的文件
  • The App can only access files you explicitly choose
  • 无法访问未授权的文件或目录
  • Cannot access unauthorized files or directories

4.2 Wi-Fi 传输相关的网络访问4.2 Network Access for Wi-Fi Transfer

Wi-Fi 导入方式需要使用：

Wi-Fi import requires:

  • 网络访问权限（INTERNET）
  • Network Access Permission (INTERNET)
  • 本地局域网通信
  • Local Network Communication

用途：用于建立手机与电脑之间的局域网传输通道，使 TXT 文件能通过 Wi-Fi 导入到手机。

Purpose: To establish a local network transfer channel between phone and computer, enabling TXT files to be imported to the phone via Wi-Fi.

我们不会利用网络权限上传数据、连接外部服务器、收集网络信息。

We do not use network permissions to upload data, connect to external servers, or collect network information.

5. 第三方服务5. Third-Party Services

5.1 Firebase Analytics（Google）5.1 Firebase Analytics (Google)

本应用使用 Google 提供的 Firebase Analytics 服务收集匿名使用统计数据，用于了解应用功能的使用情况，从而改进用户体验。

The App uses Firebase Analytics (provided by Google) to collect anonymous usage statistics, helping us understand how features are used and improve the user experience.

Firebase Analytics 可能收集的信息包括：

Firebase Analytics may collect:

  • 应用版本号
  • App version
  • 设备型号及操作系统版本
  • Device model and OS version
  • 设备语言与国家/地区（基于 IP 地址，不精确到个人）
  • Device language and country/region (based on IP address, not precise location)
  • 匿名的应用使用事件（如导入文件、进入阅读器、阅读时长、功能使用偏好）
  • Anonymous app usage events (e.g., file import, opening the reader, reading duration, feature usage preferences)
  • Firebase 自动生成的匿名应用实例标识符（App Instance ID），用于统计分析，不与个人身份关联
  • An anonymous App Instance ID automatically generated by Firebase, used solely for analytics and not linked to personal identity

Firebase Analytics 不会收集：

Firebase Analytics does not collect:

  • 您的 TXT 文件内容
  • Your TXT file contents
  • 您的姓名、邮箱、电话等个人身份信息
  • Your name, email, phone number, or other personal identity information
  • 精确的 GPS 位置信息
  • Precise GPS location

以上数据由 Google 处理，受 Google 隐私政策约束。本应用收集数据仅用于改善应用体验，不用于广告定向或出售给第三方。所有分析数据均通过 HTTPS 加密传输。

This data is processed by Google and subject to the Google Privacy Policy. Data collected by the App is used solely to improve the app experience and is never used for ad targeting or sold to third parties. All analytics data is transmitted via HTTPS encryption.

您可以通过系统设置中的"隐私"或"广告"选项，限制广告追踪或重置广告 ID，从而减少 Google Analytics 的数据关联。

You may reduce Google Analytics data association by limiting ad tracking or resetting your Advertising ID through your device's Privacy or Ads settings.
