1. 服务内容1. Service Description

本应用是一款本地 TXT 内容阅读工具，为用户提供：

This App is a local TXT content reading tool that provides users with:

  • 系统文件选择器（SAF）导入本地 TXT 文件
  • System file picker (SAF) to import local TXT files
  • 在设备本地阅读、解析、翻页、设置调整等功能
  • Local reading, parsing, page turning and settings adjustments features
  • 通过 Wi-Fi 局域网方式导入 TXT 文件（手机与电脑在同一局域网下）
  • Import TXT files via Wi-Fi local network (phone and computer on the same local network)
  • 所有文件处理操作均在设备本地完成，不上传至服务器
  • All file processing operations are completed locally on the device and not uploaded to servers

本应用不提供云端同步功能，也不传输数据至外部服务器。

This App does not provide cloud sync functionality and does not transmit data to external servers.
